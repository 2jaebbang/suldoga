# suldoga
### 한성대학교 멋쟁이사자처럼 4팀

## How to start
### Clone project
```git clone https://github.com/pressogh/suldoga.git```
### Virtualenv Setting
1. ```cd suldoga```
2. ```python -m venv venv```
3.
- #### Windows
```.\venv\Scripts\activate```
- #### Mac
```./venv/bin/activate```
### Install packages
```pip install -r requirements.txt```
### Start server
```python manage.py runserver```
